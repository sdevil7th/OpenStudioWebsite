Scriptable Automation

Go beyond manual drawing. Write TypeScript scripts to generate complex modulation, algorithmic transitions, and generative patterns directly in the automation lane.

automation.ts
// Generate sine automation
const track = Session.getTrack(1);
for(let i=0; i<100; i++) {
  track.automation.addPoint({
    time: i * 0.1,
    value: Math.sin(i / 10),
    type: 'bezier'
  });
}
